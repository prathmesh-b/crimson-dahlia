# crimson-dahlia
Repository for Team Crimson Dahlia - Spring 2026 Cohort

## Community Aid

**The idea:** A simple, accessible web app where community members can ask for help and offer help to each other. Someone needs groceries picked up? A ride to an appointment? Help paying a utility bill? This is where neighbors can connect.

The key word here is **_accessible_** â€” this should be easy for anyone to use, regardless of their tech savviness. Think Craigslist vibes: straightforward, no-frills, gets the job done. How do you make it easy for someone to post a request, and easy for helpers to find requests they can fulfill?


## Minimum Viable Product (**Goal by 2/4/2026**)
- Make an account
- Make a help post
  - Delete post
  - Edit post
  - Mark post as completed
- Browse a feed
- Enter a zipcode to see posts

### Automated Flow
- Person A posts
   - _Task is available_

- Person B accepts the offer
   - _Task is in progress_

- Person A marks the task as complete
  - _Task is complete (then deleted)_
